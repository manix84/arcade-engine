# 🌱 Code Of Conduct

Arcade Engine is a small personal project, but the same basic standard applies:
be clear, respectful, and practical.

## ✅ Expected Behavior

- Be respectful of different experience levels and backgrounds.
- Keep technical disagreement focused on the code and the outcome.
- Give useful context when reporting bugs or proposing changes.
- Accept maintainers' decisions about scope, timing, and project direction.
- Avoid wasting other contributors' time with low-effort or bad-faith
  interactions.

## 🚫 Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Sexualized language or imagery.
- Publishing private information without permission.
- Repeatedly derailing issues or pull requests.
- Spam, scams, or abusive automation.

## 🧭 Enforcement

The maintainer may edit, hide, close, lock, or delete comments, issues, pull
requests, or discussions that violate this code of conduct.

The maintainer may also block users from participating in the project when
needed to protect the project and its contributors.

## 📬 Reporting

Report problems privately to the project maintainer when possible. Include
links, screenshots, timestamps, or other context that helps explain what
happened.

Reports will be handled with reasonable discretion.
